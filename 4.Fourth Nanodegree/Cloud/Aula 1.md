# Aula 1

* Cluster: É um conjunto de computadores interconectados que trabalham para oferecer maior desempenho e disponibilidade.
  * Da pra alocar clusters para uma empresa só (sublocar).
  * Foi por causa dessa sublocação que surgiu a AWS, pois começaram a usar o equipamento que ficava ocioso.
* Cloud -> Data Center -> Cluster -> Linux
* Data center tem o corredores quentes e frios.
  * Saídas de servidor é quente, já a frente é a parte fria.
* Data center precisa ser replicação, ou seja, um data center em outro local com o backup do data center principal.
