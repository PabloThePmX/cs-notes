# Aula 5

* Existem diversos tipos de servidores.
  * Web, de aplicação, banco de dados, DNS, DHCP, VoIP, etc.
* SSH (Secure Shell)
  * É um protocolo de rede que foi projetado para permitir uma conexão segura e criptografada entre dois computadores através de uma rede não confiável (internet).
  * Usa uma chave.
  * Túneis SSH.

## Servidor WEB
* Alguns exemplos são o `Nginx` e `Apache`.
* O Nginx possui proxy embutido.

## Docker
* Surgiu em 2013.
* Os volumes são mecanismos que permitem que os containers acessem e compartilhem dados de forma persistente e independente do ciclo de vida do container.
* As redes permitem a exposição do container para o mundo exterior, ou criar a rede interna entre eles.

### Docker Engine
* Docker Daemon é o serviço de background que controla os containers.
* Docker CLI é a interface que permite que usuários interajam com o docker.
