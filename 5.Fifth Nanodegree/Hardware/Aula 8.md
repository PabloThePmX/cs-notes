# Aula 8

* Na hora de instalar o SO na Pi, definir o hostname e habilitar o SSH.
* Para acessar o ssh, utilizar o `ssh hostname@remote_ip`.
* Para descobrir o ip na Pi, utilizar `ip r` ou `hostname -I`.
  * Ip do router é o necessário para o SSH.
* Para setar um ip fixo, utilizar `sudo nmtui`.
  * Network Text User Interface.
  * Vai abrir um menu para alterar a configuração da conexão.
